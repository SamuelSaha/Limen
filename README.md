# LIMEN

Human-matched housing options for uncertain travelers.

Travelers with flexible dates pay a coordination fee (€50–€100) to get matched with verified hosts. Matches are scored on date overlap, trust signals, and swap potential.

## Status

**Phase 2 — Code foundation.** Transitioning from the [no-code concierge MVP](./docs/PRD.md) to a Nuxt 3 fullstack application.

## Key Concepts

### Matching Algorithm

The core product logic lives in `server/utils/matching.ts`. Scores are 0–100 based on:

| Signal | Weight | Rationale |
|---|---|---|
| Date flexibility overlap | 50% | No overlap = no match |
| Verification quality | 30% | Trust is the product |
| Reciprocity (swap potential) | 20% | Two-way swaps are the defensible use case |

### Coordination Tiers

| Tier | Fee | Conditions |
|---|---|---|
| Standard | €50 | ±3 days flexibility |
| Premium | €100 | ±7 days, priority matching |
| Swap | €75 | Two-way exchange |
